Vue Php Mysql SimpleCRUD 
=============

Install and run
-------
- Download code
```console
git clone https://github.com/idvazquezrivera/vue-crud-php-puro.git
```
- Move code to server

Apache
```console
mv vue-crud-php-puro /var/www/
```

MAMP
```console
mv vue-crud-php-puro /Applications/MAMP/htdocs/
```
